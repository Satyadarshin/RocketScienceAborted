About the project

All of the output you see here was written in modern HTML, CSS, and vanilla (ES6) javaScript. It’s an open-ended project, which increases the sophistication of output simply as a learning outcome.

The project was developed in Firefox and makes negligible concessions to broader browser support (I make no attempt to support Internet Explorer). Clearly this isn’t acceptable in a production project, but to emphasise, this is a self-training sandbox, so if something doesn’t work, or looks clumsy, that’s within project scope.

About the data

Because this is a training project, I’ve restricted myself to browser-native JavaScript. The award lists are stored as regular JSON files, not in a database.
